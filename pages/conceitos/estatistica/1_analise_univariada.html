def mediana(valores):
    ordem = sorted(list(valores))
    if len(ordem) % 2 == 0:
        meio = int(np.floor(len(ordem) / 2) - 1)
        meio1 = meio + 1
        mediana = (ordem[meio] + ordem[meio1]) / 2
    else:
        meio = int(np.floor(len(ordem) / 2))
        mediana = ordem[meio]
    return mediana

Quantis¶

Segundo Bussab e Morettin (2010) tanto a m√©dia como o desvio padr√£o podem n√£o ser medidas adequadas para representar um conjunto de dados, pois:

  1. s√£o afetados, de forma exagerada, por valores extremos;
  2. apenas com estes dois valores n√£o temos id√©ia da simetria ou assimetria da distribui√ß√£o dos dados.

Uma medida que pode ser considerada no lugar √© o quantil.

Podemo definir uma medida, chamada quantil de ordem p ou p-quantil, indicada por q(p) onde p √© uma propor√ß√£o qualquer, $0 < p < 1$, tal que 100p% das observa√ß√µes sejam menores do que q(p). Os quantis mais famosos s√£o os que veremos no box plots, que s√£o:

$$\text{q(0,25)} = q_1 \text{: 1¬∫ Quartil = 25¬∫ Percentil} $$$$\text{q(0,50)} = q_2 \text{: Mediana = 2¬∫ Quartil = 50¬∫ Percentil} $$$$\text{q(0,75)} = q_3 \text{: 3¬∫ Quartil = 75¬∫ Percentil} $$

Mostraremos como se c√°lcula com um exemplo:

Exemplo: Temos o seguinte conjunto de valores: $10, 25, 23, 34, 8, 2, 7$

  • 1¬∫ Passo: colocar os valores em ordem crescente: $2, 7, 8, 10, 23, 25, 34$

  • 2¬∫ Passo: Achar a mediana (2¬∫ Quartil) dos valores. Como os valores s√£o impares, a mediana ser√° dada por $x_{\frac{n+1}{2}}$. Como $n$ √© 7, $n+1$ ser√° 8, $\frac{8}{2}=4$. Queremos o valor que est√° na posi√ß√£o 4, que em nosso caso √© o 10.

  • 3¬∫ Achando a mediana, agora temos dois conjuntos de dados, um abaixo da mediana e um acima. O conjunto abaixo √©: $2, 7, 8$. E o acima √©: $23, 25, 34$. Agora devemos achar a mediana de cada conjunto para acharmos o 1¬∫ e o 3¬∫ quartil respectivamente. Com isso temos que o 1¬∫ Quartil √© 7 e o 3¬∫ Quartil √© 25.

Quando o conjunto de valores tiver um n de tamanho par, usamos o c√°lculo da mediana quando n for par, que √© dado por $\frac{x_{\frac{n}{2}} + x_{\frac{n}{2}+1}}{2}$

Exemplo: Temos o seguinte conjunto de valores: $10, 25, 23, 34, 8, 2, 7, 40$

  • 1¬∫ Passo: colocar os valores em ordem crescente: $2, 7, 8, 10, 23, 25, 34, 40$

  • 2¬∫ Passo: Achar a mediana (2¬∫ Quartil) dos valores. Como os valores s√£o pares, a mediana ser√° dada por $x_{\frac{n}{2}} + x_{\frac{n}{2}+1}$. Como $n$ √© 8, os valores ser√£o 4 e 5. Queremos os valores que est√£o na posi√ß√£o 4 e 5, que em nosso caso s√£o os valores 10 e 23. Calculando a mediana deles temos: $\frac{10 + 23}{2} = 16,5$

  • 4¬∫ Achando a mediana, agora temos dois conjuntos de dados, um abaixo da mediana e um acima. O conjunto abaixo √©: $2, 7, 8, 10$. E o acima √©: $23, 25, 34, 40$. Agora devemos achar a mediana de cada conjunto para acharmos o 1¬∫ e o 3¬∫ quartil respectivamente. Com isso temos que o 1¬∫ Quartil √© 7,5 e o 3¬∫ Quartil √© 29,5.

Um segundo m√©todo de achar os quartis √© pela formula: $$Q_j = x_k + (\frac{j(n+1)}{4}-k)(x_{k+1}-x_k)$$ Onde:

  • $j = 1, 2$ e $3$
  • $k$ √© o maior inteiro menor ou igual a $\frac{j(n+1)}{4}$
  • $n$ √© o n√∫mero de elementos do conjunto

Pegaremos o primeiro exemplo para usar a f√≥rmula:

Valores: $10,25,23,34,8,2,7$

Valores ordenados: $2, 7, 8, 10, 23, 25, 34$

Vamos calcular o primeiro quartil:

sabemos que $\frac{j(n+1)}{4} = \frac{1(7+1)}{4} = \frac{8}{4} = 2$

Logo $k$ ser√° igual a $2$. Substituindo na equa√ß√£o temos

$Q_1 = x_2 + (2-2)(x_{2+1}-x_2)$

$Q_1 = x_2 + (0)(x_{3}-x_2)$

$x_2$ = 7, vamos substituir

$Q_1 = 7 + (0)(8-7)$

$Q_1 = 7 + (0)(1)$

$Q_1 = 7$

A simetria nos diz aonde nosso conjunto de dados est√° mais concentrado.

  • Distribui√ß√µs Assim√©tricas: Quando a distribui√ß√£o for assim√©trica, nos temos valores diferentes para moda, m√©dia e mediana ($\overline{x} = M_d = M_o$)

    • Assim√©tria √† esquerda (ou negativa): Quando os valores mais baixos das observa√ß√µes s√£o predominantes, tendo uma cauda mais longa para esquerda,nesse caso a media √© menor que a mediana que √© menor que a moda ($\overline{x} < M_d < M_o$)

Sua f√≥rmula √© dada por: $$\text{assimetria} = M_3 = \frac{\sum_{i=1}^{n}(x_i - \overline{x})¬≥}{n}$$

Uma desvantagem de utilizar a f√≥rmula acima est√° na depend√™ncia da unidade de medida dos dados. Se tivermos trabalhando com dados em reais, a assimetria retornar√° reais ao cubo.

$$\text{coeficiente momento de assimetria} = \alpha_3 = \frac{M_3}{s¬≥}$$

  • Se coeficiente momento de assimetria > 0, ent√£o assimetria a direita (assimetria positiva)
  • Se coeficiente momento de assimetria < 0, ent√£o assimetria a esquerda (assimetria negativa)

def coeficiente_assimetria(valores):
    media = sum(valores) / len(valores)
    assimetria = sum((valores - media)**3) / len(valores)
    desvio_padrao = np.sqrt(sum((valores - media)**2) / (len(valores) - 1))
    return assimetria / (desvio_padrao ** 3)

Curtose √© o grau de achatamento de uma distribui√ß√£o em rela√ß√£o a uma distribui√ß√£o padr√£o. Essa medida mede o agrupamento de valores da distribui√ß√£o em torno do centro. Quanto mais agrupados em torno do centro, maior ser√° o valor da curtose.

  • Leptoc√∫rtica: √â quando a medida de curtose √© maior do que a da distribui√ß√£o normal. Nesse caso a mais valores no centro

$$\text{coeficiente momento de curtose} = \alpha_4 = \frac{M_4}{s‚Å¥}$$
